Datos generales de la materia

Materia: Desarrollo Web I

Docente: Ing. Guillermo Calderón

La asignatura introduce los fundamentos del desarrollo web moderno, desde la estructura y sintaxis de HTML y CSS hasta el diseño adaptativo con media queries. Los contenidos incluyen: modelo de caja, selectores avanzados, posicionamiento, visualización, maquetación fija y fluida, y principios de diseño responsive para diferentes dispositivos.

  • Guía 1: Introducción a HTML y CSS.
  • Guía 2: Selectores CSS avanzados y modelo de caja.
  • Guía 3: Posicionamiento, visualización y maquetación.
  • Guía 4: Introducción al responsive design.

Índice de guías y ejercicios

Guía 1 — Introducción a HTML y CSS

Estructura básica de documentos HTML y aplicación de estilos con CSS, utilizando estilos embebidos, la etiqueta <style> y hojas de estilo externas.

  • Ejercicio 1 — Estilos embebidos: Colorear fondo, alinear y formatear encabezados y párrafos. Ver
  • Ejercicio 2 — Etiqueta <style>: Definir reglas CSS internas para tipografía, contenedores y clases personalizadas. Ver
  • Ejercicio 3 — CSS externo: Enlazar un archivo .css y aplicar estilos globales. Ver

Guía 2 — Selectores avanzados y modelo de caja

Uso de selectores descendientes, de hijos, adyacentes y de atributos; pseudo-clases y pseudo-elementos, y aplicación del modelo de cajas (márgenes, bordes, rellenos).

  • Ejercicio 1 — Sitio “El discurso del rey”: Aplicar selectores avanzados, enlaces con estados y efectos con :hover, :first-letter y :first-line. Ver
  • Ejercicio 2 — Tablero de damas: Construir un tablero 8×8 con :nth-child() alternando colores. Ver
  • Ejercicio 3 - Biografía: Biografía de Iván Alessandro Vásquez llevada a cabo con los pasos. Ver

Guía 3 — Posicionamiento, visualización y maquetación

Práctica con position, display, float, overflow y z-index para construir layouts fijos y fluidos, además de un menú horizontal multinivel.

  • Ejercicio 1 — Nominaciones a mejor película: Tres columnas con imágenes centradas, corrección de desvinculación por flotados y ejemplos de posicionamiento relativo/absoluto. Ver
  • Ejercicio 2 — Menú horizontal multinivel: Listas anidadas, ocultamiento/visualización con display y efecto :hover. Ver
  • Ejercicio 3 — Agencia de viajes: Plantilla con encabezado, contenido y galería usando maquetación fija y luego fluida. Ver

Guía 4 — Responsive Design

Adaptación de estilos a móvil, tablet y desktop mediante media queries, estructuras fluidas, tamaños relativos y hojas de estilo por dispositivo.

  • Ejercicio 1 — Media queries: Definir puntos de quiebre para modificar colores, distribución y tipografías. Ver
  • Ejercicio 2 — Layout responsivo: Versión móvil, tablet y escritorio del sitio de agencia de viajes con hojas separadas. Ver

FODA — Desempeño en las guías

Fortalezas

  • Estructuración semántica con HTML5 y estilos coherentes con CSS3.
  • Dominio del modelo de caja y corrección de problemas con flotados.
  • Manejo básico de media queries para vistas móvil y tablet.

Oportunidades

  • Aplicación de los conceptos en sitios reales y portafolios profesionales.
  • Explorar herramientas adicionales para prototipado y accesibilidad.
  • Refuerzo de UI/UX para mejorar la experiencia en dispositivos diversos.

Debilidades

  • Profundizar en posicionamiento complejo y efectos avanzados.
  • Perfeccionar la consistencia tipográfica y espaciados en layouts largos.
  • Optimizar imágenes y recursos para rendimiento.

Amenazas

  • Rápida evolución de tecnologías y expectativas de la industria.
  • Limitaciones de tiempo para prácticas y mejoras continuas.
  • Diferencias de soporte entre navegadores y dispositivos.
