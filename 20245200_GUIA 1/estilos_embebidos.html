Estilos embebidos en las etiquetas HTML

Usar estilos CSS embebidos en las etiquetas HTML es el peor y menos utilizado para agregar reglas de estilo. No es nada recomendable por la dofiocultad que implica realizar un cambio a posteriori.

Esta forma de incluir CSS directamente en los elementos HTML solamente se utiliza en determijnadas situaciones en las que se deben incluir un estilo muy especoifico para un solo elemento en concreto.

Para insertar un estilo CSS en un elemento se utiliza la propiedad de HTML, que en ningun momento se debe confundir con la etiquetas <style> que se introduce en la cabecera del documento
